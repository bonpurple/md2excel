# Release Notes - v2.1.3 (2026-03-26)

## Fixed
- 閉じていない強調記法が、行内で不正に装飾として解釈される不具合を修正しました。
  - `a*b`
  - `*c`
  - `_f`
  - `g**h`
  - `**i`
  - `__l`
- 未解決の `*` / `_` / `**` / `__` が文字として出力されず、後続文字だけに斜体・太字が適用される場合がある問題を修正しました。
- CommonMark の解釈に合わせ、同一行内で閉じていない強調記法はリテラル文字列として出力されるよう修正しました。

## Changed
- 未解決の強調開始記号を次行継続として扱う条件を見直しました。
- 行末のハード改行または `<br>` によって継続が明示されている場合にのみ、開いている強調状態を次行へ引き継ぐようにしました。

## Notes
- 以下のような見出し内文字列は、従来どおりそのまま表示されます。
  - `p_q_`
  - `v__w__`
- 以下のような正しく閉じた強調記法は、従来どおり装飾が適用されます。
  - `s**t**` → `st` が太字
