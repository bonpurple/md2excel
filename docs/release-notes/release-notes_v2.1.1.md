# Release Notes - v2.1.1 (2026-03-10)

## 概要
v2.1.1 では、リスト継続行の出力処理を修正しました。  
これまで、箇条書き項目の継続行が同一セルに追記され、1セル内で連結される場合がありました。  
本バージョンでは、継続行を縦方向の新規行として出力するよう変更し、より自然で読みやすい Excel 出力に改善しています。

## 修正内容
- リスト継続行の追記処理を見直しました。
- 継続行を既存セルへ追記する方式から、縦方向の新規行として出力する方式に変更しました。
- 箇条書きの説明行・継続行が1セルにまとまってしまうケースを改善しました。

## 修正例

入力 Markdown:

```md
## title

* test  
  text  
  text
```

修正前:

* C4: `text text`

修正後:

* C4: `text`
* C5: `text`

## 影響

* 箇条書き項目の継続行が、Excel 上で行単位に分かれて表示されるようになります。
* 従来よりも、設計書・仕様書の読みやすさと視認性が向上します。
